<h1>Zeri BOT</h1>

![image](https://github.com/cauahfb/Zeri-MusicBot/assets/100867839/718e9422-aee8-412a-bb51-5e5a953b1bbb)

<p>Um bot criado inspirado na Zeri, com funcionalidades para reprodução de música no Discord.</p>
<p>Zeri BOT possui falas ácidas e ariscas, não sendo recomendado o uso para todos.</p>

<h3>Funções</h3>

<ul>
<li>/help - Exibe os comandos que podem ser utilizados.</li>
<li>/play - Procura e Reproduz uma música do Youtube, caso já tenha alguma música em reprodução será adicionada a lista de músicas.</li>
<li>/queue - Apresenta a lista de próximas músicas a serem tocadas.</li>
<li>/skip - Pula para a próxima música.</li>
<li>/stop - Para a reprodução de músicas.</li>
</ul>
